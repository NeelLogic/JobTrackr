# JobTrackr Vision

Students often apply to many internships, co-ops, and entry-level jobs. It becomes difficult to track application status, deadlines, referrals, interviews, and follow-ups.

JobTrackr helps students manage job applications in one place.

## Target Users
- Students
- New graduates
- Job seekers

## Core Features
- Add job application
- View all applications
- Update application status
- Track deadlines
- Add notes
- Store referral contact
