# JobTrackr Requirements

## Project Overview

JobTrackr is a web application designed to help students and job seekers track internship, co-op, and full-time job applications in one place.

## Target User

- Students
- New Graduates
- Job Seekers

## MVP Features

- Add Application
- View Applications
- Edit Application
- Delete Application
- Track Application Status
- Add Notes
- Store Referral Information
- Store Job Posting Link

## Pages

### Dashboard
Shows a summary of job applications.

### Applications
Shows all saved job applications.

### Add Application
Form used to add a new job application.

### Edit Application
Form used to update an existing job application.

## Application Fields

| Field | Example |
|---------|---------|
| Company | RBC |
| Role | Software Developer Intern |
| Location | Toronto |
| Status | Applied |
| Date Applied | 2026-06-02 |
| Deadline | 2026-06-15 |
| Job Link | Posting URL |
| Notes | Follow up next week |
| Referral Name | Sachin |
