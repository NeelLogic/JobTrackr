# JobTrackr

JobTrackr is a full-stack job application tracking system built to help students organize internship, co-op, and entry-level job applications.

## Features
- Add and manage job applications
- Track application status
- Store company, role, date applied, and deadline
- Add notes and referral details
- View all applications in one dashboard

## Tech Stack
- Spring Boot
- Angular
- MySQL
- Git/GitHub
